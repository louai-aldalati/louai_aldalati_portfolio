من أنا؟

طالب هندسة برمجيات أمتلك خبرة عملية في تطوير تطبيقات الويب (Django) وبناء أنظمة الذكاء الاصطناعي. أعمل على ابتكار حلول متكاملة تدمج تقنيات الواجهات الخلفية (Backend) مع النماذج اللغوية (LLMs) والرؤية الحاسوبية. أجيد ثلاث لغات بطلاقة، وأتميز بالتفكير التحليلي والقدرة على العمل ضمن بيئات متنوعة لحل مشكلات برمجية معقدة.

المشاريع التقنية

  • مساعد ذكي مخصص للشركات (AI Hotel Chatbot)
    التقنيات: Llama 3 (8B Instruct), RAG, Fine-Tuning, NLP
    • تطوير مساعد ذكي متقدم يعتمد على النموذج اللغوي مفتوح المصدر Llama 3 لتقديم ردود دقيقة ومؤتمتة.
    • تطبيق تقنيات التوليد المعزز بالاسترجاع (RAG) والضبط الدقيق (Fine-Tuning) لتخصيص إجابات النموذج بشكل كامل بناءً على قواعد وبيانات وسياسات فندق محدد، مما رفع من كفاءة خدمة العملاء.
    مشاهدة على GitHub
  • نظام التعرف على تعابير الوجه في الوقت الفعلي
    التقنيات: Computer Vision, MobileNetV2, Transfer Learning, Python
    • بناء نظام رؤية حاسوبية محلي يحلل البث المباشر لكاميرا الويب لتصنيف 8 مشاعر إنسانية مختلفة (مثل السعادة، الحزن، الغضب) بدقة عالية.
    • استخدام تقنية نقل التعلم (Transfer Learning) عبر الاعتماد على معمارية النموذج المسبق التدريب MobileNetV2، وإعادة تدريب الطبقات النهائية (Fine-tuning) لتسريع الأداء وتقليل استهلاك الموارد.
    مشاهدة على GitHub
  • نظام التنبؤ بالطقس (Weather Forecasting ML API)
    التقنيات: Machine Learning, XGBoost, Time-Series Analysis, Django, Django REST Framework
    • تصميم وتدريب نموذج تعلم آلة باستخدام خوارزمية XGBoost لتحليل البيانات الزمنية (درجة الحرارة، الرطوبة، اتجاه الرياح) والتنبؤ بالقيم المستقبلية.
    • تطوير الواجهة الخلفية للنظام باستخدام إطار عمل Django، وتقديم نموذج الذكاء الاصطناعي كخدمة برمجية متكاملة عبر واجهة برمجة تطبيقات (Django REST API) لسهولة ربطه مع تطبيقات الويب أو الموبايل.
    مشاهدة على GitHub
  • منصة تعلم لغات تعتمد على التكرار المتباعد (Kelime6)
    التقنيات: Django, Python, Relational Database, Git, Scrum, Agile
    • برمجة نظام تعليمي يعتمد على خوارزمية التكرار المتباعد (SRS) لجدولة ظهور الكلمات زمنياً بناءً على دقة أداء المستخدم.
    • تطوير وحدات متكاملة تشمل: مصادقة المستخدمين، لوحات تقارير تحليلية، ونظام اختبارات ديناميكي يتضمن لعبة (Wordle) التفاعلية.
    • إدارة التطوير بمنهجية Agile/Scrum، مع الالتزام الصارم بمعايير (KISS) لتجنب الـ Code Smells، وإدارة التحديثات عبر GitHub.
    مشاهدة على GitHub
  • نظام إدارة الحجوزات العقارية (TinyHouse Management System)
    التقنيات: Django, Python, Relational Database, SQL (Stored Procedures, Triggers, Functions), RBAC
    • تطوير منصة لإدارة العقارات تدعم نظام صلاحيات متقدم (RBAC) لثلاث فئات مختلفة: مدراء (Admins)، أصحاب عقارات (Hosts)، ومستأجرين (Guests).
    • هندسة قاعدة بيانات علائقية (Normalized)، مع توظيف متقدم للـ SQL عبر (Stored Procedures) و (Triggers) لأتمتة تحديثات الحجز والدفع.
    • بناء ميزات أساسية شملت: الفلترة المتقدمة، تتبع التوافر الزمني، إدارة طلبات الحجز والتقييمات، وتوليد التقارير المالية.
    مشاهدة على GitHub
